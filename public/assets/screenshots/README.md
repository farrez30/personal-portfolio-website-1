# Web project screenshots

Drop one screenshot per **web** project here, named by the project's `id`
(see `components/portfolio/portfolioData.js`). Recommended: 16:10, ~1280×800,
JPG or PNG, optimized.

Expected files:

- `raja-roti-cemerlang.jpg`
- `uruzin.jpg`
- `coderoach-studio.jpg`
- `tumtim-cookies.jpg`
- `locascore.jpg`
- `viralytics.jpg`
- `rpn-agro.jpg`
- `groseria.jpg`

Until a file exists, the card and modal gracefully fall back to a gradient
"poster" with the project title — so missing screenshots never show a broken
image. To use a different filename/extension, update that project's
`thumbnail` field in `portfolioData.js`.
